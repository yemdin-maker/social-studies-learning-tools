# U.S. History Help Learning Tools

This repository contains the web tools embedded on `ushistoryhelp.weebly.com`.

## Published pages

- `/tracker/` - Social Studies Learning Tracker
- `/arcade/` - Learning Rewards Arcade
- `/scheduling/` - Tutoring session scheduling form

## Google Apps Script

The `apps-script/` directory stores the source for the tracker/rewards backend and the tutoring booking backend. Updating these files in GitHub does not deploy them automatically to Google Apps Script.

## Source files

The root copies preserve the original Weebly embed snippets. The page directories contain the GitHub Pages entry points.
